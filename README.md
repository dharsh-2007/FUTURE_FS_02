# Mini CRM System

This project is a lightweight Customer Relationship Management (CRM) application designed to organize and manage client leads efficiently.

The system allows administrators to view, track, and update lead information, helping streamline communication and improve follow-up processes. It focuses on core CRM functionalities commonly used in real-world business environments.

## Highlights

- Centralized lead management  
- Clear lead status tracking  
- Simple and structured data handling  
- Practical implementation of full-stack concepts  

This project demonstrates the ability to build and manage a functional web application with real business use cases.
